jQuery HTML Demo

.html()

Summary

The 
.html()
function from jQuery is both a getter and a setter function:
  • Get: Get the HTML contents of the first element in the set of matched elements.
  • Set: Set the HTML contents of every matched element.

GET: .html() -- returns string

Description: Get the HTML contents of the first element in the set of matched elements.
  • This method does not take any arguments.

SET: .html( htmlString ) -- returns jQuery object

Description: Set the HTML contents of every matched element.
  • .html( htmlString ) -- htmlString input
    htmlString: A string of HTML to set as the content of each matched element.

  • .html( Function(Integer index, htmlString oldhtml) ) -- Function input (renders to htmlString value)
    Function: A function returning the HTML content to set. Receives the index position of the element in the set and the old HTML value as arguments. jQuery empties the element before calling the function; use the oldhtml argument to reference the previous content. Within the function, 
    this
    refers to the current element in the set.
