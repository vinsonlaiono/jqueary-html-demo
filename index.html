.html()

Using jQuery in your HTML document, you can use the .html() function to:

  • get the HTML contents of the first element in the set of matched elements, or
  • set the HTML contents of every matched element.

Using .html() as a getter

In an HTML document, .html() can be used to get the contents of any element. If the selector matches more than one element, only the first match will have its HTML content returned. Take a look at this code:

$( "div.container" ).html();

In order to get the <div>'s content, it would have to be the first one with class="container" in the document:

<div class="container">
    <div class="box">Sample Box</div>
</div>

The function would return the following:

<div class="box">Sample Box</div>

This method uses the browser's innerHTML property.

Using .html() as a setter

When .html() is used to set an element's content, any content that was in that element is replaced by your new content.

Take a look at the following HTML:

<div class="container">
    <div class="box">Sample Box</div>
</div>

The content of <div class="container"> can be set like this:

$( "div.container" ).html( "<p>My new content.</p>" );

That line of code will replace everything inside <div class="container">:

<div class="container">
    <p>My new content.</p>
</div>
